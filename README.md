# XPC Skill Card

## Overview

Skill cards capture the clinical experience of a medical practitioner to build a clearer picture of the range and depth of microskills they have demonstrated within a particular condition or domain.

Rather than testing knowledge, skill cards track real-world exposure: for a given condition (e.g. Type 2 Diabetes), has this practitioner ever managed it with insulin? Have they documented a foot exam? Have they ordered a UACR or screened for diabetic retinopathy?

Each item on a skill card represents a discrete clinical microskill tied to actual patient encounters, giving a longitudinal view of capability as it develops over time.
